# Aiti Guru — Управление товарами

Тестовое задание: SPA для авторизации и управления товарами.

## Стек

React 18, TypeScript (strict), Vite, Tailwind CSS, Zustand, React Router 6, Axios

## Запуск

```bash
npm install
npm run dev
```

Откроется на `http://localhost:5173`

## Тестовый вход

Используются учётные данные DummyJSON API:

| Логин    | Пароль   |
|----------|----------|
| emilys   | emilyspass |

Полный список: https://dummyjson.com/users

## Сборка

```bash
npm run build
npm run preview
```
